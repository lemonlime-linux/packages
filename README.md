# Lemonlime Linux Packages

Definitions, build scripts, and specifications for core and supplemental packages in Lemonlime Linux.

## About
Maintains source definitions, patches, recipes, and packaging automation that form the system’s software base.

## What’s Included
- Build files and metadata  
- Packaging automation and update workflows  
- Tracking for upstream and community packages  

## Contributing
Packaging guidelines are available in the documentation repository. Contributions from packagers, testers, and developers are welcome.
